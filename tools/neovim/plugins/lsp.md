# LSP & Diagnostics

Activated when language support (LSP) is active.

| Shortcut | Action |
| :--- | :--- |
| `grd` | **Goto Definition** |
| `grr` | **Goto References** |
| `gri` | **Goto Implementation** |
| `grD` | **Goto Declaration** |
| `grt` | **Goto Type Definition** |
| `grn` | **Rename** variable everywhere |
| `gra` | Open code actions (**Goto Code Action**) |
| `gO` | Open **Document Symbols** (current file) |
| `gW` | Open **Workspace Symbols** (entire project) |
| `<leader>th` | Toggle Type Inlay Hints |
| `K` | Show documentation on hover |
| `[d` / `]d` | Previous / Next diagnostic |
