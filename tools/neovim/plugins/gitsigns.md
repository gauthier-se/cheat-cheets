# Gitsigns (Git Micro-View)

| Shortcut | Action |
| :--- | :--- |
| `]c` | Jump to **next** Git modification |
| `[c` | Jump to **previous** Git modification |
| `<leader>hp` | **Preview hunk** |
| `<leader>hs` | **Stage hunk** (prepare for commit) |
| `<leader>hr` | **Reset hunk** (discard modification) |
| `<leader>hS` | Stage entire current file |
| `<leader>hu` | Undo staging |
| `<leader>hR` | Reset all modifications in current file |
| `<leader>hb` | Show line author ("Blame") |
| `<leader>tb` | Toggle "Blame" display at end of line |
