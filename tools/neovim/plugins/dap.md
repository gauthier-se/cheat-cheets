# Debugging (DAP)

Allows tracing and debugging executions.

| Shortcut | Action |
| :--- | :--- |
| `<F5>` | Start/Continue debugging |
| `<F1>` | Step Into |
| `<F2>` | Step Over |
| `<F3>` | Step Out |
| `<leader>b` | Toggle **Breakpoint** |
| `<leader>B` | Toggle **Conditional Breakpoint** |
| `<F7>` | View result of last session (UI) |
