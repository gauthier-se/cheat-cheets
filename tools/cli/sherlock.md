# sherlock — Social Media Recon

Hunt down social media accounts by username across social networks.

```bash
sherlock <username>
```

## Examples

```bash
# 1. Search for a username across all supported sites
sherlock user123

# 2. Search for multiple usernames
sherlock user1 user2 user3

# 3. Output results to a specific CSV file
sherlock user123 --csv
```

## Installation

```bash
# Using pipx (recommended)
pipx install sherlock-project

# Or via Homebrew
brew install sherlock
```
